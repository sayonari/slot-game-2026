# SPEC - 技術仕様・要件定義

作成日：2026-06-11 ／ 根拠：PLAN.md ＋ ユーザーヒアリング

## コンセプト
ド派手で中毒性のあるWebスロットゲーム。
**ベースはビデオスロット型（5リール×3段）＋ パチスロ的な激アツ演出・目押しミニゲームを盛り込んだハイブリッド型。**

## 機能要件

### 1. コアゲーム（スロット本体）
- 5リール×3段のビデオスロット
- 複数ペイライン（20ライン程度）＋ スキャッター判定
- ベット額変更（コイン制、賭博性なしの娯楽用仮想コイン）
- オートスピン、クイックストップ（タップでリール即停止＝擬似目押し）
- RTP（還元率）を内部パラメータとして調整可能な設計にする

### 2. ド派手演出（最重要要件）
- リーチ・激アツ予告：リール枠の発光色変化（青→緑→赤→虹）、カットイン演出
- フリースピン突入演出、大量WIN時の「BIG WIN / MEGA WIN / JACKPOT」フルスクリーン演出
- パーティクル（コイン噴出、花火、稲妻、紙吹雪）を多用
- 画面シェイク、フラッシュ、スローモーション停止などの演出
- 笑える演出・バカ演出（外れ時の脱力演出、超レアなネタ演出）も混在させる

### 3. 繰り返し遊べる要素
- フリースピンモード（マルチプライヤー付き）
- ジャックポット（プログレッシブ：プレイするほど積み上がる）
- レベル・経験値システム（レベルアップで新シンボル・新演出・新マシンが解放）
- 実績（アチーブメント）システム
- シンボル・演出のコレクション図鑑
- デイリーボーナス（日替わりコイン配布、連続ログイン要素）
- コインが尽きた場合の救済（時間経過 or ミニゲームで復活）

### 4. 隠し要素
- 隠しコマンド（例：コナミコマンド）で特殊モード発動
- 超低確率のシークレット演出・シークレット実績
- 特定条件（時刻、回数など）で発動するイースターエッグ

### 5. データ保存
- `localStorage` を使用（cookieより容量・APIが適切。サーバー不要）
- 保存対象：コイン残高、レベル、実績、図鑑、設定、デイリーボーナス受領状況
- エクスポート／インポート（文字列コピー）でバックアップ可能にする

## 非機能要件
- **静的サイトで完結**（GitHub Pages で公開、サーバーサイド処理なし）
- PC・スマホ両対応（レスポンシブ、タッチ操作対応）
- 高フレームレート（60fps目標）、スマホでも快適に動作する負荷設計
- 音はON/OFF・音量調整可能。ブラウザの自動再生制限に対応（初回タップで解禁）

## 技術構成
- **言語／ビルド**：TypeScript ＋ Vite（静的ビルド）
- **グラフィック**：Three.js（WebGL）による3Dリール・パーティクル演出。UIはHTML/CSSオーバーレイ
- **サウンド**：Web Audio API によるコード生成（BGM・効果音とも合成で作成。著作権リスクゼロ）
- **素材方針**：コード生成中心。シンボル等はプロシージャル生成（SVG/Canvas/3Dモデル）
- **デプロイ**：GitHub Actions → GitHub Pages（リポジトリはパブリック化）
- **保存**：localStorage

## 追加仕様（2026-06-12 第2サイクル：ユーザー要望）

### 1. 当たりやすさの向上（楽しさ優先）
- 全スピンにハズレ時1回の引き直し（実質ヒット率大幅UP）
- 天井を8連敗→5連敗に短縮
- WILD・BONUSシンボルの出現重みを増加（フリースピン突入 約1/45へ）

### 2. ボーナスゲーム（MINI / MINOR / MAJOR）
- 通常スピンごとに約1/40でボーナス抽選
- ランク抽選：MINI 60%（ベット×30）／MINOR 30%（×80）／MAJOR 10%（×250）
- 演出：ランクルーレット → ランク確定 → メダル大量獲得演出
- 獲得後「⚡LUCKY TIME」突入：10回転メダル消費なし＆超高確率（引き直し5回）で自動回転

### 3. パチンコ的演出の追加
- 🐠 魚群予告（大チャンス示唆、たまにガセ）
- 🔁 疑似連（仮停止→「N連!!」→再回転、最大2連）
- ⚡ キュイン確定音＋金色フラッシュ（ボーナス確定時）
- 図鑑・実績にも対応項目を追加
